# 📊 Customer Product EDA Analysis

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on customer and product datasets using Python.

The objective is to analyze inventory values, discounts, customer purchasing behavior, and generate business insights.

---

## 🛠 Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

```text
Customer-Product-EDA-Analysis/
│
├── Customers.csv

└── README.md
```

---

## 📋 Tasks Performed

- Data Loading and Exploration
- Inventory Value Calculation
- Discount Analysis
- Product Counting
- Customer Filtering
- Top Cities Visualization
- Product Purchase Analysis
- Customer Purchase Reporting
- Discounted Purchase Analysis
